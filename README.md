# Next.js E-Commerce Application

This is a simple e-commerce application built using **Next.js (App Router)** and **Tailwind CSS** as part of an interview assignment.

## Features
- Product Listing Page (PLP)
- Product Detail Page (PDP)
- Global Cart using React Context
- Add / Remove / Update Cart items
- Server-side data fetching
- Responsive design

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Fake Store API

## Project Structure
- `app/` – Pages (PLP, PDP, Cart, Home)
- `components/` – Reusable UI components
- `context/` – Cart global state
- `lib/` – API fetch utilities

npm run dev
